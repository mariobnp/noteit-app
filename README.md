# NoteIt - Personal Notes App 📝

**NoteIt** adalah aplikasi manajemen catatan pribadi berbasis web yang dibangun menggunakan **React**. Aplikasi ini memungkinkan pengguna untuk membuat, mengarsipkan, mencari, hingga menghapus catatan dengan integrasi RESTful API.

Aplikasi ini dilengkapi dengan fitur autentikasi, manajemen tema (Dark/Light mode), serta dukungan multi-bahasa (Indonesia & Inggris).

## ✨ Fitur Utama
* **Autentikasi Pengguna**: Register dan Login untuk menjaga privasi catatan.
* **Manajemen Catatan**: Tambah, hapus, dan arsipkan catatan secara real-time.
* **Pencarian**: Temukan catatan dengan cepat berdasarkan judul.
* **Dukungan Multi-bahasa**: Beralih antara Bahasa Indonesia dan Inggris.
* **Mode Gelap/Terang**: Antarmuka yang adaptif dan nyaman di mata menggunakan React Context.
* **Responsif**: Tampilan optimal di perangkat mobile maupun desktop menggunakan Tailwind CSS.

## 🛠️ Teknologi yang Digunakan
* **React.js** (Functional Components & Hooks)
* **React Router v6** (Routing & Protected Routes)
* **React Context** (State Management untuk Tema & Bahasa)
* **Tailwind CSS** (Styling)
* **SweetAlert2** (Notifikasi Interaktif)
* **RESTful API** (Data Persistence)
